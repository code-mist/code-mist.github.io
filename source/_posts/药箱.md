# 药箱

### **解热镇痛类**

用于发烧、头痛、牙痛、肌肉酸痛等。

1. **对乙酰氨基酚**（扑热息痛）：适合儿童、成人，对胃肠道刺激小。
2. **布洛芬**：退烧镇痛，抗炎效果好（肠胃不适者慎用）。*儿童需备对应剂型（滴剂/混悬液），按体重计算剂量。*如果有慢性肝病，首选。
3. 阿司匹林：除肚疼外都能用，但刺激胃，饭后服。**儿童病毒感染时禁用阿司匹林**（可能引发瑞氏综合征，致死率高）。且与对乙酰氨基酚/布洛芬同服会增加胃肠道出血风险。**16/18岁以下禁用**



### **感冒与过敏类**

1. **生理性海水鼻腔喷雾**：缓解鼻塞、鼻腔干燥。
2. **抗组胺药**：**氯雷他定**或**西替利嗪**：缓解过敏性鼻炎、荨麻疹等（嗜睡副作用小）。
3. **止咳祛痰**

- **盐酸氨溴索**或**乙酰半胱氨酸**：化痰。
- **蜂蜜**（1岁以上儿童）：缓解夜间咳嗽（天然有效）。



### **肠胃不适类**

1. **口服补液盐Ⅲ**：腹泻时预防脱水（尤其儿童和老人）。
2. **蒙脱石散**：急性腹泻时临时止泻（需与补液盐配合）。肠炎
3. 多潘立酮片(吗丁啉) ：恶心呕吐、消化不良、腹胀。心脏疾病患者慎用，可能引起心律失常。
4. **益生菌**（如双歧杆菌）：调理肠道菌群。
5. 乳果糖：便秘，安全性好，增加大便水分。
6. **抗酸药/胃黏膜保护剂**：**铝碳酸镁**：缓解胃痛、烧心。
7. 开塞露：便秘，外用，另有防止皮肤开裂用途。

### 抗过敏药

- 扑尔敏：荨麻疹、皮炎、虫蚊等皮肤过敏

- 氟雷他定：二代组胺，**通常较少嗜睡；少数情况下可能出现心脏相关副作用（如心律相关问题），多见于特定风险人群或剂量/合并用药情况下。**

- 三代组胺

  > 地氟雷他定，商品名芙必叮（推荐）
  >
  > 盐酸左西替利嗪

> **建议：家庭常备1种即可。** 推荐**氯雷他定**（嗜睡副作用小，成人儿童都可用），其他可移除。

### **皮肤问题类**

1. **抗过敏止痒**：**炉甘石洗剂**：痱子、蚊虫叮咬、湿疹止痒（皮肤破损勿用）。
2. **抗真菌药**：**咪康唑乳膏**：脚气、股癣。
3. **弱效激素药膏**：**氢化可的松乳膏（1%）**：轻度湿疹、皮炎（短期使用）。
4. **蚊虫叮咬/止痒**：**风油精**、**薄荷脑软膏**（避开眼、口及破损皮肤）。

> 抗生素类--消炎用：
>
> - 莫匹罗星软膏：脓疮、水泡、蚊虫叮咬、割伤、烧伤
> - 红霉素眼膏：眼周感染，在伤口处涂抹也有预防伤口感染作用





### 抗生素类

- 阿莫西林：呼吸道、软组织、泌尿感染

  禁忌： 1、青霉素过敏、淋巴瘤患者禁用； 2、哮喘、湿疹、疱疹病史慎用； 3、与庆大、环丙、培氟禁忌；

- **阿莫西林 + 头孢类**：同属β-内酰胺类，存在交叉过敏风险

- 诺氟沙星：三代喹诺酮类，肠炎痢疾、呼吸道、生殖感染

  禁忌： 1、氟喹诺酮类药过敏者禁用 2、肝肾功能减退者，慎用 3、<u>**18岁以下禁用（影响软骨发育）**</u>

- 新诺明：磺胺类 尿路，中耳炎，腹泻，卡氏肺孢子虫肺炎

  禁忌： 1、**阻止叶酸的代谢，贫血患者禁用**； 2、重度肝肾功能损害禁用； 3、临床常见病原菌对该品常呈现耐药

- 阿奇霉素：十五元大环内酯类 支气管、肺炎、咽炎、中耳炎、扁桃体、皮肤感染

  禁忌： 1、对阿奇霉素、红霉素或其他任何大环内酯类药物过敏者禁用 2、和地高辛、三唑仑合作时密切观察

- 头孢类抗生素：药禁酒。 建议备第四代、第五代头孢。如考虑成本因素，则建议备多备第三代，少备第五代。

  ```markdown
  1. 一代头孢:头孢拉定、头孢唑林、头孢氨苄、头孢羟氨苄--G+菌强，G-菌弱，肾毒性较大
  2. 二代头孢:头孢呋辛、头孢孟多、头孢替啶、头孢克洛、头孢丙烯--G+菌弱，G-菌弱，肾毒性稍小;
  3. 三代头孢:头孢噻肟、头孢克肟、头孢泊肟、头孢他啶、头孢哌酮、头孢曲松--G+菌弱，G-菌弱，基本无肾毒。
  4. 四代头孢:头孢吡肟、头孢匹罗G+菌G-菌都较强
  5. 五代头孢:头孢洛林、头孢托罗、头孢吡普--超广谱，G+G-均较强，无肾毒性
  ```

- 甲硝唑：厌氧菌感染，腹腔、消化道、软组织、骨和关节、破伤风、口腔

  禁忌： 1、中枢神经系统疾患和血液病者禁用。 2、妊娠期妇女及哺乳期妇女禁用。 3、肝中蓄积，肝功不全者慎用。 4、**2017年WHO列为2B类致癌物，且孕期/哺乳期禁用**

> ==**建议：保留1-2种外用抗生素（如莫匹罗星软膏）即可，口服抗生素全部移除，需要时去医院或药店现买。**==

### 维生素类

> 综合维生素-品牌很多，可以考虑同步补充钙、镁、锌
>
> **建议：改为1瓶复合维生素（如善存/21金维他）**，既省空间又避免单一过量。

- 维生素A：也叫胡萝卜素，增强对传染病抵抗力，来源广泛【【缺乏会导致：眼睛干燥畏光，多泪、视觉含糊、皮肤干粗】
- 维生素B1：影响消化、神经，增强抗病能力；来自米糠、花生米、胡桃、蚕豆、酵母片等。【【【缺乏会导致：手脚发麻、气色欠安、消化不良】
- 维生素B2：维持神经、消化器官和视觉器官的健康，来自动物肝、蛋黄、卷心菜、菠菜和萝卜。【【【缺乏会导致：黑白炎、舌炎、唇炎、眼炎、皮肤炎症、继发性血虚。
- 维生素B6：促进氨基酸和脂肪代谢；来自酵母、肝、蛋奶、豆类、花生。【【【缺乏会导致：肌肉痉挛，外伤不愈合】
- 维生素B12：抗恶性贫血维生素，来自动物肝脏、肉蛋奶、鱼，植物中一般没有。【【【缺乏会导致：食欲不振、乏力/精神差、皮肤苍白、注意力下降、可能出现手脚麻木或行走不稳。
- 维生素C：也叫抗坏血酸，维持细胞间联络组织正常机能；来自柠檬、橘子、菠菜、番茄、茶叶等。【【【缺乏会导致：牙龈出血、牙齿松动、毛细血管脆性增加、皮下出血、瘀斑，口腔和消化器官黏膜出血、坏血病（也可带“发育不良/体重增长差”）】
- 维生素D：增进人体的钙化，促使齿、骨正常发育；来自鱼肝油、蛋黄、动物肝、牛奶等。【【【缺乏会导致：头部多汗，儿童易患佝偻病，成人骨质软化。】



### 紧急救护类

**内服**

```markdown
硝酸甘油片：冠心病、心绞痛。是处方药，且有效期极短（开封后通常只有3-6个月）
	禁忌：
	1. 、禁用于心肌梗塞早期(有严重低血压及心动过速时)严重
	2、贫血、青光眼、颅内压增高和已知对硝酸甘油过敏的患者
速效救心丸：活气血，祛瘀止痛，增加冠脉血流量，缓解心绞痛。
	禁忌：不明确
```

**外用**

```markdown
高锰酸钾消毒片:**浓度配制错误极易导致化学烧伤。家庭使用风险高，建议移除或改为碘伏。**
	用于：
	1. 皮炎或急性湿疹，伴继发感染的湿敷，清洗小面积溃疡		2. 消毒、杀菌、治疗真菌如脚气
	不可内服。加水溶解配制使用。随用随配。
    水浅红色时用于消毒；
    水深红色时用于杀菌；
    水紫红色时 治疗真菌

云南白药：中国神药，出血类。内有保险丸一粒可紧急内服。**孕妇禁用，且保险子（红色小丸）是严重出血时急救用，不能随意内服。**
三角巾：无菌包装，比纱布用途广泛。
碘伏：广谱杀菌，可灭杀细菌繁殖体、真菌、原虫和部分病毒。可用于皮肤、粘膜；也可处理烫伤、滴虫性阴道炎、霉菌性阴道炎、皮肤霉菌感染。
```

### **外伤护理类**

1. **消毒抗菌**

- 碘伏棉签/溶液（伤口消毒，刺激性小）。
- 75%医用酒精（皮肤消毒，伤口慎用）。

1. **包扎用品**：无菌纱布、创可贴、医用胶带、棉签。
2. **烫伤**：**湿润烧伤膏**或**磺胺嘧啶银乳膏**（小面积烫伤应急）。



### **其他实用物品**

1. **工具类**：电子体温计（更安全）、镊子、剪刀、一次性手套。
2. **防护类**：口罩、医用冷敷贴（物理退热/扭伤冷敷）。

| 物品                       | 用途                               |
| -------------------------- | ---------------------------------- |
| **医用检查手套**           | 处理伤口时保护双方                 |
| **LED手电筒**              | 检查口腔、耳道、瞳孔               |
| **急救联系卡**             | 写上家属电话、过敏史、血型、慢性病 |
| **生理盐水（小支装）**     | 冲洗伤口、眼睛，比自制高锰酸钾安全 |
| **不同规格创可贴**         | 指尖、关节、大面积等               |
| **电子血压计**（如有老人） | 比单纯备硝酸甘油更实用             |
| **退热贴/冰袋**            | 物理降温，与药物配合               |



### **注意事项**



1. **每3-6个月检查一次有效期**，尤其是硝酸甘油、眼药水、胰岛素等对温度敏感的药物
2. **分格存放**：儿童药与成人药分开、外用药与内服药分开
3. **保留说明书**：不要只留药板，说明书上有禁忌和剂量
4. **避光干燥**：不要放在浴室（潮湿）或阳台（高温）
5. **及时就医原则**：高烧3天不退、胸痛、呼吸困难、严重外伤、意识模糊——药箱只是过渡，不是终点

------

> ⚠️ **免责声明**：以上检查基于家庭药箱管理常识，不构成医疗建议。涉及处方药（如抗生素、硝酸甘油）的使用，请务必咨询医生或药师。

- 不滥用抗生素（需医生指导）。
- 孕妇、哺乳期、慢性病患者用药前咨询医生。
- 家庭药箱旨在应对**轻微症状**和**临时处理**

**及时就医**：高烧持续不退、严重外伤、呼吸困难、腹痛剧烈等，立即就医
