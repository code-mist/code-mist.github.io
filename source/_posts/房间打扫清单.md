# 房间打扫清单

## 一、打扫顺序与常用清洁剂

### 打扫三原则

**从上到下**：天花板→墙壁→柜顶→桌面→地面。灰尘只会往下落，顺序乱了就是反复做无用功。

**从干到湿**：先扫/吸走浮尘，再湿布擦。浮尘遇水成泥浆，越擦越花。

**从里到外**：从房间最里侧往门口方向推进，垃圾最后一起带走，不走回头路。

**频率铁律**：每天随手5分钟 > 每周重点清洁 > 每月深度保洁。干净靠的是不让脏污过夜，不是靠大扫除。

### 常用清洁剂速查

| 清洁剂       | 用途                             | 注意事项                                 |
| :----------- | :------------------------------- | :--------------------------------------- |
| 洗洁精       | 厨房油污、一般台面               | 选A类食品接触级                          |
| 小苏打       | 去油、除味、吸湿、调糊去污       | 万能选手                                 |
| 白醋         | 除水垢、杀菌、软化污渍           | 不可用于大理石/天然石材                  |
| 柠檬酸       | 水垢专用（花洒、水龙头、电水壶） | 食品级，比白醋效率高                     |
| 重油污清洁剂 | 油烟机、灶台、烤箱               | 喷后静置5分钟再擦                        |
| 洁厕剂       | 马桶黄垢/尿渍                    | 不可与84混用                             |
| 84消毒液     | 地面消毒、除霉斑                 | 绝对不可与酸性清洁剂混用（产生剧毒氯气） |
| 玻璃清洁剂   | 玻璃、镜子                       | 洗洁精水可替代                           |

## 三、通用打扫分类（多个房间共用）

### 1. 桌面与台面

**打扫方法**：每天睡前花2分钟把台面杂物归位，顺手用干布擦掉水渍/食物渣。每周用超细纤维布（微湿）沿同一方向擦拭台面，再换干布收干水痕。每月彻底清空台面所有物品，全面擦拭包括摆件底部；抽屉翻一遍，过期无用的扔掉。

**使用工具**：超细纤维布、旧牙刷（清理边角）

**频率**：每日随手归位，每周擦一遍，每月彻底清空一次。


### 2. 地面

**打扫方法**：每日扫把贴地“弓”字形路线扫，沿踢脚线走，玄关、餐桌下、厨房水槽前是重点。每周先扫/吸尘，再拖地；温水加几滴洗洁精，最后一遍清水拖净；拖地顺序：卧室→客厅→厨房→卫生间，不同区域换拖布。每月移开家具清扫底部，整屋踢脚线湿布擦一遍。

**使用工具**：扫把/吸尘器、平板拖把、洗洁精、温水

**频率**：每日扫重点区域，每周全屋拖一次，每月移家具深度清。


### 3. 窗户与窗帘

**打扫方法**：每月擦玻璃内侧，先用湿布去浮尘，再用玻璃刮+洗洁精水刮净，干布[擦眼镜的那种]收边；窗框滑道用旧牙刷+吸尘器窄吸头清理；纱窗能拆则拆下冲淋+软毛刷，阴干装回。每季度洗窗帘（布艺按标签洗涤，纱帘轻柔手洗/装袋机洗，百叶窗逐片湿擦）。高层外侧用长柄工具或请专业保洁，安全第一。

**使用工具**：玻璃刮、旧牙刷、吸尘器窄吸头、洗洁精、软毛刷

**频率**：每月擦内侧+清滑道+洗纱窗，每季度洗窗帘。


### 4. 开关面板与门把手

**打扫方法**：每周用酒精棉片或微湿超细纤维布擦拭，注意边角缝隙的灰尘。开关面板周围的墙壁容易脏，顺手擦掉手印。

**使用工具**：酒精棉片、超细纤维布

**频率**：每周擦一次。


## 四、卧室

### 1. 床上用品

**打扫方法**：起床后被子翻面铺开散潮气，洗漱后再叠；粘毛滚滚一遍床单去毛发皮屑。每周床单被套枕套用60℃以上热水洗；枕芯拍打+太阳暴晒2-3小时；床垫撒小苏打静置30分钟后吸尘器吸净。每季被芯晾晒（蚕丝/羽绒阴干，棉被可暴晒）；床垫头尾调换；枕芯1-2年换新，被芯3-5年换新。

**使用工具**：粘毛滚、吸尘器、小苏打

**频率**：每日散潮气+滚毛，每周换洗+除螨，每季深度养护。


### 2. 衣柜

**打扫方法**：每月清空衣物，柜内用微湿布擦拭隔板和底板，柜门内外擦净。换季时整理衣物，过季的洗净晾干后收纳（箱底垫旧报纸+放樟木条/干燥剂），一年未穿的捐赠或丢弃。柜顶每月除尘掸扫灰。

**使用工具**：超细纤维布、静电除尘掸、收纳箱、樟木条/干燥剂

**频率**：每月擦柜内+柜顶，换季深度整理。


### 3. 床头柜

**打扫方法**：每日归位杂物（水杯、眼镜、书本），随手擦水渍。每周清空擦拭台面，注意底部角落。

**使用工具**：超细纤维布

**频率**：每日归位，每周擦拭。


## 五、客厅

### 1. 沙发

**打扫方法**：每周坐垫抬起，吸尘器吸缝隙（食物残渣、毛发、皮屑）；布艺沙发用微湿布擦拭表面。每月沙发底部吸尘；布艺沙发套可拆洗则按标签洗。每季皮质沙发上护理油保养，布艺沙发用专用清洁剂局部去渍+整体蒸汽清洁。地毯每周吸尘，每月局部去渍，每半年专业清洗一次。

**使用工具**：吸尘器（窄吸头）、超细纤维布、皮革护理油/布艺清洁剂

**频率**：每周吸缝隙，每月吸底部+去渍，每季深度保养。


### 2. 电视柜与茶几

**打扫方法**：每天顺手归位杂物。每周清空台面，微湿布擦拭整个表面，再干布收水痕；抽屉内部同样清理。电视屏幕每月用专用清洁剂+超细纤维布轻擦（关机后），切忌酒精/纸巾。

**使用工具**：超细纤维布、屏幕清洁剂、旧牙刷（边角）

**频率**：每日归位，每周擦拭，每月擦屏幕。


### 3. 窗帘与窗户

见“通用打扫分类-窗户与窗帘”。


## 六、厨房

### 1. 灶台与油烟机

**打扫方法**：每天做完饭趁灶台有余温湿布一擦；油烟机表面顺手擦。每周喷重油污清洁剂，静置5分钟后用百洁布/海绵刷洗，清水擦净；油杯拆下倒掉油污+洗洁精泡洗。每月油烟机滤网拆下泡重油污剂后刷洗，灶具火盖取下刷洗（注意疏通火孔）。

**使用工具**：重油污清洁剂、百洁布、旧牙刷、橡胶手套

**频率**：每日随手擦，每周重点清，每月深度拆洗。


### 2. 水槽与下水道

**打扫方法**：每天食物残渣随手捞起扔垃圾桶，不冲下水道。每周小苏打+白醋倒入下水口，冒泡后热水冲淋除味；水槽内壁用百洁布+洗洁精刷洗，不锈钢水槽每月用不锈钢清洁膏抛光一次。地漏防臭芯每月拆下刷洗。

**使用工具**：小苏打、白醋、百洁布、不锈钢清洁膏

**频率**：每日捞残渣，每周冲管道+刷水槽，每月清地漏芯+抛光。


### 3. 台面

见“通用打扫分类-桌面与台面”。厨房台面额外注意油污，每周用洗洁精水重点擦拭，调料瓶底部和瓶身油垢每月彻底清理一次。


### 4. 冰箱

**打扫方法**：每周清点一次，过期食品扔掉。每月彻底清空，温水+小苏打擦拭内壁隔板；密封条用酒精棉片仔细擦拭（防李斯特菌）；冰箱外部用微湿布擦净；背后冷凝器每半年清灰。

**使用工具**：小苏打、温水、酒精棉片、超细纤维布

**频率**：每周清点，每月彻底清内壁+密封条，每半年清冷凝器。


### 5. 微波炉与烤箱

**打扫方法**：每周微波炉放碗水+几片柠檬，高火加热3-5分钟，蒸汽软化油污后一擦即净。烤箱每季度小苏打调糊涂抹内壁（避开发热管），静置30分钟后湿布擦净；烤盘泡洗洁精水后刷洗。

**使用工具**：柠檬、小苏打、湿布

**频率**：每周微波炉蒸汽清洁，每季度烤箱深度清。


### 6. 地面

见“通用打扫分类-地面”。厨房地面额外注意每日拖掉油渍水渍，每周用洗洁精水重点拖洗。


## 七、卫生间

### 1. 马桶

**打扫方法**：每周洁厕剂喷内壁（尤其出水口边缘），静置后马桶刷刷洗，冲净；马桶盖、座圈、外壁、底座用84稀释液或洁厕剂湿布擦净。每月底座背后和地面接缝处同样清洁。

**使用工具**：洁厕剂、84消毒液（仅外部）、马桶刷、橡胶手套

**频率**：每周全擦，每月深度清底座背后。


### 2. 洗手台与水龙头

**打扫方法**：每天用完顺手擦干台面水渍和牙膏渍。每周微湿布全面擦拭，玻璃用玻璃刮处理。每月柠檬酸溶液浸泡水龙头出水口+擦拭表面除水垢；下水口拆开清毛发。

**使用工具**：柠檬酸、玻璃刮、旧牙刷

**频率**：每日顺手擦干，每周全面擦，每月除水垢+清下水口。


### 3. 淋浴区/浴缸

**打扫方法**：每次洗完澡玻璃刮刮掉墙面和玻璃门水珠，减少水垢堆积。每周喷玻璃清洁剂/白醋水擦玻璃门；墙面瓷砖用洗洁精水刷洗。每月花洒头拆下浸泡柠檬酸除水垢；地漏拆开清毛发+倒小苏打白醋除味。

**使用工具**：玻璃刮、玻璃清洁剂/白醋、柠檬酸、旧牙刷、小苏打

**频率**：每次刮水，每周擦墙+玻璃，每月清花洒+地漏。


### 4. 镜子

**打扫方法**：每周用玻璃清洁剂喷后玻璃刮刮净，干布收边角；白醋水可替代清洁剂。

**使用工具**：玻璃刮、玻璃清洁剂

**频率**：每周擦一次。


### 5. 地面

见“通用打扫分类-地面”。卫生间地面特别注意：用84稀释液拖地杀菌，最后一遍清水拖净。地漏周边重点清洁，每月用84拖一次。


## 八、阳台

### 1. 地面与栏杆

**打扫方法**：每周扫除落叶尘土，湿布擦栏杆扶手。每月刷洗地砖（硬毛刷+洗洁精水），地漏清杂物。

**使用工具**：扫把、硬毛刷、洗洁精

**频率**：每周扫+擦栏杆，每月刷地+清地漏。


### 2. 洗衣机

**打扫方法**：每月洗衣机槽清洁剂倒入，高温模式空洗一遍；橡胶密封圈用湿布擦干（避免积水发霉）；洗涤剂盒拆下冲洗。每季度机身外部擦净，背后通风口清灰。

**使用工具**：洗衣机槽清洁剂、旧牙刷

**频率**：每月洗内筒+擦密封圈，每季度清外部+通风口。


## 九、家电专项（独立于房间）

### 1. 空调

**打扫方法**：每月拆下滤网，花洒冲洗+软毛刷轻刷，阴干后装回。每季度室内机面板用微湿布擦拭（先断电）；室外机周围清杂物，散热片用空调清洁剂喷洗（按说明书操作）。

**使用工具**：空调清洁剂、软毛刷

**频率**：每月洗滤网，每季度清散热片。


### 2. 电视机/显示器

见“客厅-电视柜与茶几”部分。每月用专用清洁剂+超细纤维布，关机后轻擦，切忌液体渗入边框。

**使用工具**：屏幕清洁剂、超细纤维布


### 3. 路由器/机顶盒

**打扫方法**：每季度断电后用干布擦拭，旧牙刷清理散热孔积灰（积灰导致散热不良、信号衰减甚至死机）。

**使用工具**：干布、旧牙刷

**频率**：每季度清一次。

**频率**：每年一次。

## 十、衣服鞋类



---

### 洗衣服

#### 日常洗衣基本原则

**颜色分开**：深色和浅色分开洗，新买的深色衣服前几次单独洗，避免染色。

**材质分开**：牛仔裤、毛巾、外套这类厚的和薄的T恤、衬衫分开洗——厚的不容易洗干净，薄的经不起搓。

**看洗标**：衣服领子或侧缝有个小标签，上面画着水温、能不能漂白、能不能烘干，照着来就行。

---

#### 脏了怎么处理

**油渍**（吃饭溅的油、菜汤）：洗之前用洗洁精直接涂在油渍上，干搓一下，放5分钟再正常洗。洗洁精去油比洗衣液强。

**血渍**（小伤口蹭的）：**冷水**泡，别用热水（一烫蛋白质凝固就再也洗不掉了）。肥皂或内衣皂搓一下，基本能掉。

**汗渍发黄**（白T恤领口腋下）：小苏打+温水泡半小时再洗。或者牙膏搓黄渍，牙膏里的研磨剂能蹭掉表层氧化。

**笔印/墨水**：酒精棉片/75%酒精倒在棉片上敷一会儿再搓，溶解后正常洗。

**咖啡/茶渍**：趁湿的时候用热水冲一下再正常洗，干了就难了。

**红酒渍**：用白酒/白醋倒在渍上搓一下，把颜色溶出来再洗。

**衣服洗完还有怪味**：水温不够或泡太久，洗完没及时晾，细菌发酵了。加点白醋重洗一遍，洗完立刻晾。

---

#### 洗衣液怎么放

**滚筒洗衣机**：洗衣液倒进洗涤剂槽（拉开抽屉第一个格子），消毒液和柔顺剂倒对应的格子，别混。

**波轮洗衣机**：先放水，再放洗衣液，搅匀了再放衣服。别直接把洗衣液倒衣服上——高浓度接触布料容易褪色。

**用多少量**：洗衣液的瓶盖上有刻度线，按那个来。倒多了没用，漂洗不干净反而刺激皮肤。

---

#### 内裤袜子怎么洗

**内裤**：手洗最好，专用内衣皂搓，洗完用热水烫一下。机洗的话单独洗，别和外衣混。三个月左右换一批新的。

**袜子**：单独洗，别和内裤一起。汗味重的先泡一下再洗。

**毛巾**：每周用热水（60℃以上）加洗衣液洗一次，别用柔顺剂（柔顺剂会让毛巾不吸水）。每月一次高温空洗洗衣机除菌。

---

#### 哪些东西不能放进洗衣机

- **真丝**：手洗，凉水，不能用碱性皂，专用真丝洗涤剂。别拧，平铺阴干。
- **羊毛衫**：凉水手洗，别搓，轻轻按压。别甩干，一甩就缩水。平铺阴干。
- **羽绒服**：可以机洗，但要用羽绒专用模式或轻柔模式，别甩太久，容易炸绒。

---

#### 平时穿多久洗一次

| 品类             | 频率                                      |
| :--------------- | :---------------------------------------- |
| T恤/衬衫         | 穿1次就洗（汗渍细菌多）                   |
| 裤子（非牛仔裤） | 穿2-3次洗一次                             |
| 牛仔裤           | 穿4-5次再洗（不是越勤越好，容易变形褪色） |
| 外套             | 一个月左右，脏了随时洗                    |
| 毛衣/针织衫      | 穿3-4次洗一次                             |
| 内衣             | 每天换每天洗                              |
| 袜子             | 每天换每天洗                              |
| 睡衣             | 穿3-4次洗一次                             |

**核心就一句话**：沾了汗的衣服别堆，当天换当天洗。堆久了细菌发酵，洗完了也容易有闷臭味。

### 洗鞋

#### 先看鞋能不能下水

**能下水**：帆布鞋、棉布鞋、大部分网面运动鞋、普通跑步鞋。

**不能下水**：真皮鞋、翻毛皮/磨砂皮鞋、绒面鞋、带特殊涂层的鞋。这类一泡水就废。

拿不准的话，看鞋标——画着一道斜线穿过水盆的标识，就是禁止水洗。

#### 能下水的鞋（通用清洗法）

**第一步 预处理**
鞋带拆下来单独搓，鞋垫抽出来。鞋底朝上拍几下，把大块泥沙磕掉。

**第二步 刷洗**
温水（不烫手）加中性洗衣液。软毛刷蘸着刷鞋面，硬毛刷刷鞋底和外底边。网面别用力戳，会起毛。旧牙刷处理鞋舌边缘、鞋带孔这些窄缝。

**第三步 漂净**
清水冲干净泡沫，边冲边用刷子轻轻带一遍，把深层脏水带出来。多换几次水，直到完全没泡沫。

**第四步 吸水**
整个鞋用干布按一遍，吸走表面水分。鞋内塞满旧报纸团——报纸吸水，而且能把鞋撑回原形。塞实一点，别塞太胀就行。

**第五步 晾晒**
阴凉通风处自然晾干。鞋底侧放，让鞋内空气流通。

**绝对不能**：暴晒（发黄变硬）、烘干机高温烘（开胶缩水）、鞋内不塞东西直接晾（干了会塌扁变形）。

#### 不能下水的鞋

皮面和翻毛皮。

**皮面鞋**：湿布擦掉灰尘，蘸一点点中性洗洁精水擦脏的地方，擦完立刻用干布抹干。干透后上皮革护理油。

**翻毛皮/磨砂皮**：用专用绒面刷顺着一个方向梳，把灰梳掉。脏得厉害用绒面专用泡沫清洁剂，喷上后软毛刷轻刷，干布吸走脏泡沫。晾干后再把绒毛梳顺。

**共同点**：绝对不能泡水，鞋内撒小苏打除味即可。

#### 除臭（所有鞋通用）

每晚把鞋垫抽出来晾着，鞋内撒小苏打粉，第二天倒掉。核心是两双鞋轮换穿——穿一天至少晾24小时再穿，汗透了就晒。鞋柜放活性炭包或樟木条吸湿。
